# Changelog

All notable changes to DevBoard are documented in this file. The project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-20

### Added

- Bilingual GitHub profile and repository analytics dashboard
- Repository quality scoring and improvement recommendations
- Activity, contribution and language visualizations
- Side-by-side repository comparison
- PDF and Excel report exports
- Demo fallback for unavailable GitHub API responses
- Responsive dark and light themes
- Unit, UI and end-to-end test suites
- GitHub Actions and Vercel deployment configuration
